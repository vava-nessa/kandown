## Implementation Details

### MarkdownEditor Component Spec

```typescript
interface MarkdownEditorProps {
  value: string;           // markdown content
  onChange?: (val: string) => void;  // called on edit
  readOnly?: boolean;      // default false, starts in edit mode if false
  placeholder?: string;
  minHeight?: string;       // default "120px"
}
```

**Behavior:**
- Default state: `readOnly=true` → renders markdown preview only
- Click anywhere on preview → switches to `readOnly=false` → shows editor
- Toggle button (icon) in top-right corner to manually switch mode
- If `onChange` is provided, changes call it
- If `onChange` is undefined, component is read-only display

**Visual:**
- When `readOnly=true`: rendered markdown with subtle hover indicator (dashed border or eye icon)
- When `readOnly=false`: full editor toolbar visible
- Toggle button always visible in top-right corner

### Syncfusion Integration Notes

Uses `RichTextEditorComponent` with `editorMode='Markdown'` and `MarkdownEditor` service.
The `marked` library converts markdown to HTML for preview mode.
