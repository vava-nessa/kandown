# Plan

## Context
Syncfusion is a paid library. This task replaces the Syncfusion-based MarkdownEditor with [Wysimark](https://github.com/portive/wysimark), a free open-source WYSIWYG editor for Markdown that supports 100% CommonMark + GFM.

## Tech Reference
- https://github.com/portive/wysimark
- React package: `@wysimark/react`
- Docs: https://wysimark.com/docs/react
- License: Open source (free for commercial use)
