# Context

The File System Access API grants permission for a specific directory handle. This permission can be revoked by the browser (tab refresh, browser restart, user action). When a stored handle becomes invalid:

1. `verifyPermission()` at `src/lib/filesystem.ts:249-257` has **no error handling** — if the handle is revoked, `queryPermission` throws instead of returning 'denied'
2. `openRecentProject()` at `src/lib/store.ts:315-330` leaves the store in a **half-initialized state** — some state is set (projectName) but critical state (dirHandle, tasksDirHandle) remains null
3. There's **no mechanism to remove** a broken recent project entry from IndexedDB
4. `getKandownHandle()` at `src/lib/filesystem.ts:84-92` tries to create `.kandown` if it doesn't exist — if the directory was deleted AND permission was revoked, this throws uncaught

**Current flow on permission revoked:**
1. User clicks a recent project
2. `openRecentProject` calls `verifyPermission(project.handle, true)`
3. `queryPermission` throws because handle is invalid
4. Error propagates uncaught → store in half-state → no error shown
5. User sees empty board with project name but no tasks
6. No way to remove the broken entry

## How to Fix

### Step 1: Fix verifyPermission

```typescript
export async function verifyPermission(
  handle: FileSystemDirectoryHandle,
  readWrite = true
): Promise<'granted' | 'denied' | 'unavailable'> {
  try {
    const opts = { mode: readWrite ? 'readwrite' : 'read' } as const;
    if ((await handle.queryPermission(opts)) === 'granted') return 'granted';
    if ((await handle.requestPermission(opts)) === 'granted') return 'granted';
    return 'denied';
  } catch (e) {
    // Handle is invalid/revoked — can't even query permission
    console.warn('[FS] Handle is no longer valid:', e);
    return 'unavailable';
  }
}
```

### Step 2: Fix openRecentProject to be Transactional

```typescript
openRecentProject: async (project: RecentProject) => {
  const previousState = {
    dirHandle: get().dirHandle,
    tasksDirHandle: get().tasksDirHandle,
    projectName: get().projectName,
  };

  try {
    const permResult = await verifyPermission(project.handle, true);
    
    if (permResult === 'unavailable') {
      // Handle is completely invalid — offer to remove from recent
      get().toast('This project folder is no longer accessible. Removing from recent projects.', 'warning', 8000);
      await removeRecentProject(project.handle.name);
      const updated = await listRecentProjects();
      set({ recentProjects: updated });
      return;
    }
    
    if (permResult === 'denied') {
      get().toast('Permission denied — please grant access to the folder', 'error');
      return;
    }

    const kandownHandle = await getKandownHandle(project.handle);
    const tasksDir = await ensureTasksDir(kandownHandle);
    const projectName = project.handle.name;

    // All-or-nothing state update
    set({
      dirHandle: kandownHandle,
      tasksDirHandle: tasksDir,
      projectName,
      boardTitle: projectName,
    });

    await saveRecentProject({ handle: project.handle, name: projectName, lastOpened: Date.now() });
    await get().loadConfig();
    await get().reloadBoard();

  } catch (e) {
    // Rollback to previous state
    set(previousState);
    get().toast(`Failed to open project: ${(e as Error).message}`, 'error');
  }
}
```
