## Notes

Current shortcuts from the app:
- `⌘K` - Command palette
- `⌘1` / `⌘2` - Board / list view
- `N` - New task
- `R` - Reload
- `/` - Focus search
- `Esc` - Close drawer
- `⌘S` - Save current task
