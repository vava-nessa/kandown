# Context

When `kandown.json` is corrupted (invalid JSON, missing fields, wrong types), the app silently falls back to `DEFAULT_CONFIG` without telling the user. This can happen due to:

- Partial write (browser crashed mid-save)
- Manual edit mistake (user edited JSON and introduced syntax error)
- Git merge conflict left in file
- File system corruption
- Another tool overwrote the file with garbage

**Affected code:**

**Problems:**
1. `JSON.parse` failure returns `null` — treated identically to "file not found"
2. User's custom columns, theme, notifications are silently wiped
3. No indication that the config was corrupted vs simply missing
4. No backup/recovery of the corrupted config content
5. `raw.board` being `null` (JSON literal `null`) causes `...raw.board` to throw `TypeError` — not caught

## Why This Needs Fixing

**Severity: Medium**

Silent config loss means the user's board columns, theme preferences, notification settings, and field visibility all reset without explanation. For a config-heavy app, this is disorienting. The user thinks their settings "disappeared" and must reconfigure everything from scratch.

Additionally, the `...raw.board` spreading bug is a latent crash — if someone manually edits `kandown.json` and sets `board: null`, the app crashes.

## How to Fix

### Step 1: Return Result Type from readConfigFile

```typescript
export type ConfigReadResult =
  | { ok: true; config: KandownConfig }
  | { ok: false; reason: 'not_found' | 'corrupted'; rawContent?: string; error?: Error };

export async function readConfigFile(
  kandownHandle: FileSystemDirectoryHandle
): Promise<ConfigReadResult> {
  try {
    const h = await kandownHandle.getFileHandle('kandown.json');
    const file = await h.getFile();
    const text = await file.text();
    
    let raw: unknown;
    try {
      raw = JSON.parse(text);
    } catch (parseError) {
      // JSON is corrupted — return raw content for backup
      return { 
        ok: false, 
        reason: 'corrupted', 
        rawContent: text,
        error: parseError as Error 
      };
    }

    if (typeof raw !== 'object' || raw === null) {
      return { 
        ok: false, 
        reason: 'corrupted', 
        rawContent: text,
        error: new Error('Config must be a JSON object') 
      };
    }

    // Safe spread — handle null sub-objects
    const partial = raw as Record<string, unknown>;
    const merged: KandownConfig = {
      ui: { ...DEFAULT_CONFIG.ui, ...(partial.ui && typeof partial.ui === 'object' ? partial.ui : {}) },
      agent: { ...DEFAULT_CONFIG.agent, ...(partial.agent && typeof partial.agent === 'object' ? partial.agent : {}) },
      board: { 
        ...DEFAULT_CONFIG.board, 
        ...(partial.board && typeof partial.board === 'object' ? partial.board : {}),
        columns: Array.isArray((partial.board as any)?.columns) && (partial.board as any).columns.length > 0
          ? (partial.board as any).columns.filter(...)
          : DEFAULT_CONFIG.board.columns,
      },
      fields: { ...DEFAULT_CONFIG.fields, ...(partial.fields && typeof partial.fields === 'object' ? partial.fields : {}) },
      notifications: { ...DEFAULT_CONFIG.notifications, ...(partial.notifications && typeof partial.notifications === 'object' ? partial.notifications : {}) },
    };

    return { ok: true, config: merged };
  } catch (e) {
    // File not found or permission denied
    if ((e as DOMException).name === 'NotFoundError') {
      return { ok: false, reason: 'not_found' };
    }
    return { ok: false, reason: 'corrupted', error: e as Error };
  }
}
```

### Step 2: Update loadConfig with User Notification

```typescript
loadConfig: async () => {
  const { dirHandle } = get();
  if (!dirHandle) return;

  const result = await readConfigFile(dirHandle);

  if (result.ok) {
    set({ config: result.config });
    applyConfigTheme(result.config);
    return;
  }

  if (result.reason === 'corrupted') {
    // Backup corrupted content
    if (result.rawContent) {
      try {
        const backupHandle = await dirHandle.getFileHandle('kandown.json.backup', { create: true });
        const writable = await backupHandle.createWritable();
        await writable.write(result.rawContent);
        await writable.close();
      } catch {
        // Backup failed — don't block startup
      }
    }

    get().toast(
      'kandown.json is corrupted — using default settings. A backup was saved as kandown.json.backup.',
      'warning',
      10000
    );
  }

  // Use defaults for both "not found" and "corrupted"
  set({ config: DEFAULT_CONFIG });
  applyConfigTheme(DEFAULT_CONFIG);
}
```
